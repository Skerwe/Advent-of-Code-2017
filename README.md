# Advent of Code 2017

Advent of Code website: https://adventofcode.com/2017

## Build Prerequisite

* [JDK 7+](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
* [Apache Ant](http://ant.apache.org/)

## Build Commands

Compile
> ant compile

Run
> ant dist

> java -cp dist/lib/\*.jar class.name
